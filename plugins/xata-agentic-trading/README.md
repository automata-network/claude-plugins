# XATA Agentic Trading Plugin

Enable AI-powered trading on decentralized exchanges via the XATA API.

## Overview

This plugin provides natural language trading capabilities for decentralized exchanges through seamless integration with the XATA API. Interact with DEXes using conversational commands while leveraging AI to execute complex trading strategies.

## Supported Platforms

- Hyperliquid
- Lighter
- Aster

## Prerequisites

- Python 3.8 or higher
- pip package manager

## Quick Start

1. Install the plugin using Claude Code
2. Invoke the `xata-trading` skill
3. Follow the setup prompts to configure your trading environment
4. Start trading with natural language commands

## Documentation

For detailed API documentation and advanced configuration, visit the [XATA API Documentation](https://docs.ata.network/xata/api).
